# @novas/prettier-config

Prettier config


## Installation

Install this module and its `peerDependencies`, then extend it in your `.prettierrc`:

```jsonc
// package.json
{
	"devDependencies": {
		"@novas/prettier-config": "latest",
		"prettier": "2.4.1"
	}
}
```

```jsonc
// .prettierrc
"@novas/prettier-config"
```
